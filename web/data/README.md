# De-Jargonizer frequency data

`dejargonizer-bbc.csv.gz` is the BBC word-frequency dataset published with
De-Jargonizer (Rakedzon et al., PLOS ONE, 2017), under the MIT license:
<https://doi.org/10.6084/m9.figshare.5194798.v1>.

The original `BBCData.csv` SHA-256 is
`bb5678a6f616972ab58d4e18cc8a9c26d502e528adb33467696e3da045d29fa6`.
Unjargon reads it locally; no message text is sent to a detector service.
